Randall Library Projects

Virtual Shelf

  • PHP ● Yii Framework
  • MySQL ● HTML ● CSS
  • Sierra API

This project creates a local database of library inventory, indexed by call number. It involves downloading a million plus records from Sierra, the library's resource management server hosted by Innovative Interfaces Inc, using the API provided by the vendor. Since transactions with Sierra timeout after several thousand records have been received, this process has been built as a series of batch jobs that repeat every five minutes beginning at 1 in the morning. The project includes online reports for monitoring the batch jobs and viewing the resulting data. It is the foundation for a number of new applications to be built using this data.

Information Literacy

  • Responsive Design ● Drupal
  • HTML ● Sass ● CSS
  • See the Project

Information Literacy is the ability to locate, evaluate, and use information effectively. This site was created to help students, faculty, and librarians learn these skills. My contribution was to modify the theme to create Randall Library's first responsive website.

Best Bets

  • PHP ● Yii Framework
  • Bootstrap Framework
  • HTML ● CSS ● MySQL

Best Bets is a data entry app that allows library staff to key in popular terms and resources that are often searched for by students. This information will be used to improve the library's Bento Box search.

Information Kiosk Heatmap

  • JavaScript
  • Open source software

The library created an electronic information kiosk to help patrons find people, places, resources, and events within the building. We wanted to get a better idea if the application was being used the way we envisioned. ClickHeat was identified as a good open source heatmap tool to assist with this. It was my job to install the software and get it working with our application.

Follow Me Tutorials

  • JavaScript ● HTML
  • Sass ● CSS
  • Open source software
  • See the Project

This application allows librarians to create guided tutorials to help students access and use library, university, and other internet resouces. Based on the open source Guide on the Side project created by The University of Arizona, it displays instructions on one side of a browser window and a relevant web page on the other. Students are advised to use the displayed web page to follow along with the instructions provided. I was asked to customize the project for use at Randall Library, and to document the process so that other staff members could do the same.

Assignment Calculator

  • PHP ● Drupal
  • HTML ● CSS
  • Open source software
  • See the Project

The Assignment Calculator is open source software created by The University of Minnesota Libraries to help students budget their time when working on complicated projects. The student provides a due date, and the calculator breaks the assignment into a series of steps, each with its own completion date. I added additional customization requested by Randall Library, including the ability to use multiple kinds of project templates within the application for different types of assignments.
